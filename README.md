# 📊 Survey Dashboard

An interactive dashboard for visualizing survey data, built with Python, Streamlit, and Plotly.

## 🛠 Tech Stack
- Python
- Streamlit
- Plotly
- pandas

## 📋 Features
- Interactive filters (Gender, Group, Version)
- KPI cards (Respondents, Satisfaction, NPS, C1 Score)
- Satisfaction distribution chart
- Age group analysis
- NPS breakdown

## ▶️ How to Run
pip install streamlit pandas plotly openpyxl
streamlit run survey_dashboard.py
